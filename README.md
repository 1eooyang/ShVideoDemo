# ShVideoDemo
视频录制Demo   防腾讯小视频   视频压缩（FFmpeg）

#Download

Download the latest JAR or grab via Maven:
~~~xml
<dependency>
  <groupId>com.sh.shformutils</groupId>
  <artifactId>shform</artifactId>
  <version>2.0.1</version>
   <type>pom</type>
</dependency>
~~~
or Gradle:
~~~gradle
compile 'com.sh.shformutils:shform:2.0.1'
~~~
